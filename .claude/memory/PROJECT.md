# Hoja de ruta — Web Lidia Castro Fisioterapia

## Visión del proyecto

Crear una web de fisioterapia de autor, premium y elegante, que sirva primero como pitch para captar a Lidia Castro como cliente y después como su web real en producción.

## Decisiones clave tomadas

| Decisión | Valor | Motivo |
|----------|-------|--------|
| Color acento | #7A8870 (verde salvia) | Bienestar, diferenciador, alejado del ámbar |
| Tagline hero | "El arte de sanar" | Poético pero con significado claro |
| Copy | 100% español | Clínica local de Madrid |
| Tipografía | Playfair Display + Inter | Editorial + legibilidad |
| Fondo | #F9F7F2 (crema cálido) | Premium, no clínico |
